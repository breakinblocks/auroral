# Auroral 1.0.4

## New Features

- Added missing ecipe to craft the Shimmersteel Upgrade Smithing Template
- Added recipe for the Auroral Guide, crafted with a Book and a Snowball

- Auroral Nautilus now drops leather (1-2) instead of Aurora Shards

- Reduced Auroral Nautilus spawn frequency

## Bug Fixes

- Fixed Snow Angel block not rendering in world (black/purple missing texture)
- Fixed Auroral Snowlette and Auroral Nautilus spawn eggs showing missing textures
- Fixed advancement tab background not displaying correctly
