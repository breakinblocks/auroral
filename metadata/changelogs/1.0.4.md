# Auroral 1.0.4

## New Features

- Added missing recipe to craft the Shimmersteel Upgrade Smithing Template
- Added ice crafting recipe (8 Snowballs around a Water Bucket)
- Added recipe for the Auroral Guide (Book + Snowball, shapeless)

## Changes

- Auroral Nautilus now drops leather (1-2) instead of Aurora Shards
- Auroral Nautilus has a small chance to drop Prismarine Shards
- Reduced Auroral Nautilus spawn frequency

## Bug Fixes

- Fixed all crafting recipes not working due to incorrect JSON format
- Fixed spawn egg textures for Auroral Snowlette and Auroral Nautilus
- Fixed advancement tab background not displaying correctly
- Fixed duplicate resource files causing build issues
